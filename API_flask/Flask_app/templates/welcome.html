Welcome to this prediction app !

This application uses the dataset: Online Shoppers Purchasing Intention

The goal is to predict whether or not a customer will buy something on the web site according to severals features.

Variables description

Pages Value

The “PageValues” for a page are the average value of each session during which that page was visited. For the prediction, this is the most important variable.

Exit Rates

The “ExitRate” for a page is the number of times a user exit the web site though this page divided by the number of times the page was used. It’s a percentage.

Product Related

Number of pages view by the user during his session on the web site.

Product Related Duration

Time spent by the user during his session on the web site on product related pages.

Push the button to start a prediction !
